# References Directory

Save style inspiration images here. Claude uses these as visual references when generating content.

## Structure

```
references/
├── carousel-styles/          # Pinterest saves, Instagram screenshots, etc.
│   ├── clean-minimal.png     # Example: clean white carousel style
│   ├── bold-gradient.png     # Example: bold gradient with large text
│   └── ...
│
├── thumbnail-styles/         # YouTube thumbnail references
│   ├── white-bg-tech.png     # Example: white bg tech tutorial style
│   ├── dark-announcement.png # Example: dark bg announcement style
│   └── ...
│
└── README.md                 # This file
```

## How to Use

1. Save a screenshot of a carousel or thumbnail style you like
2. Drop it in the matching folder
3. When creating content, tell Claude: "Use the style from references/carousel-styles/clean-minimal.png"
4. Claude will analyze the reference and match the visual style

## Tips

- Pinterest is great for carousel layout inspiration
- Screenshot competitor thumbnails that perform well
- Name files descriptively so you can reference them easily
- Works with any image format (PNG, JPG, WebP)
