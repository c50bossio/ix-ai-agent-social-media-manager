# Landscape — 1920x1080

**Use for:** YouTube long-form, presentations, ads

---

## Layout

- Padding: 60-80px sides, 60-80px top/bottom
- Can use side-by-side layouts (illustration left, text right)
- 2-column grids work well
- More horizontal breathing room

## Typography

- Hero headline: 52-64px
- Section headline: 36-44px
- Body: 20-26px
- Labels: 14-16px

## Illustrations

- Scale up 1.2-1.5x compared to portrait
- Can place beside text instead of above

## Pacing

- More room for longer scenes (5-8s each)
- Total can be 30-60s+
- Slower transitions (18-24 frame overlap)

## Notes

- Good for data-heavy content (charts, dashboards, comparisons)
- Thumbnails and video embeds have more space
- Can show more text per screen
