# Portrait — 1080x1920

**Use for:** Reels, TikTok, YouTube Shorts, Instagram Stories

---

## Layout

- **Always center content vertically** — use `justifyContent: "center"` on the scene `<AbsoluteFill>`
- Padding: `0 52-60px` (horizontal only) — never add top/bottom padding that pushes content off-center
- Content flows vertically — stack illustration above text
- Single-column layout only (no side-by-side)
- Center illustrations horizontally
- Make elements large and bold — portrait screens are tall, fill the space

## Typography

- Hero headline: 42-52px
- Section headline: 34-36px
- Body: 17-22px
- Labels: 13-14px

## Illustrations

- Large (scene hero): scale 0.9-1.3 (roughly 180-280px)
- Medium (step icons): scale 0.55-0.6 (roughly 80-100px in circles)
- Small (inline): 28-36px fixed

## Pacing

- Total duration: 25-30s sweet spot (750-900 frames at 30fps)
- Hook scene: 3-4s (grab attention immediately)
- Content scenes: 4-5s each
- CTA scene: 3-4s
- 5-7 scenes is ideal

## Transitions

- 15 frame overlap (0.5s) — fast enough to keep energy
- Favor `fade()` for most, `slide({ direction: "from-bottom" })` for key moments

## Storytelling Structure

1. Hook — relatable pain or provocative question
2. Pain amplification — list what's wrong
3. Shift — introduce the solution
4. How it works — 3 steps max
5. Proof — show it in action
6. Value — results/stats
7. CTA — clear next step

## Duration Calculation

```
total_frames = sum(scene_durations) - (num_transitions * overlap_frames)
total_seconds = total_frames / fps
```

Example (7 scenes, 15-frame transitions):
```
120 + 120 + 90 + 150 + 150 + 120 + 120 = 870 scene frames
6 transitions * 15 frames = 90 overlap frames
870 - 90 = 780 total frames
780 / 30 = 26 seconds
```
