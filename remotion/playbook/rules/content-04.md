# Rules — Content & Storytelling

---

## Do

- Start with a hook that creates immediate emotional resonance
- Follow the arc: Problem > Solution > How > Proof > Value > CTA
- Keep text concise — max 2 lines per text block
- One core idea per scene
- End with a clear, simple call to action

## Don't

- Never cram too much text into one scene — max 2-3 short lines
- Never skip the hook — the first 2 seconds determine if someone watches
- Never end without a CTA — always tell the viewer what to do next
